Python and Kepler's Wine Barrel Problem¶

Problem: Find the dimensions of the barrel of largest volume that can be measured with rod of fixed length.

d - barrel width
R - fixed rod length
l - barrel length
vol - barrel volume

If Johannes Kepler had some modern tool like python and its incarnation, Jupyter Notebook, he could easily have visualised the solution.

Barrel volume can be viewed as function of barrel width (diameter) and as function of barrel length. Barrel dimentions calculated with both methods should be the same.

Barrel width (diameter) as argument¶

Diameter range: d = 0 .. R

From Pythagorean theorem:

$R^2 = d^2 + (\frac{l}{2})^2 = R^2 - \frac{l^2}{4}$

$l = 2\sqrt{R^2 - d^2}$

Volume as a function of barrel width (diameter):

$vol(d) = \frac{\pi d^2}{4}l = \frac{\pi d^2}{2}\sqrt{R^2 - d^2}$

The first derivative of vol(d):

$vol^{'}(d) = \frac{\pi}{2}(2d\sqrt{R^2 - d^2} - \frac{d^3}{\sqrt{R^2 - d^2}})$

Find barrel dimentions of largest volume ($vol^{'}(d) = 0$)

$\sqrt{R^2 - d^2} = \frac{d^2}{2\sqrt{R^2 - d^2}}$

$R^2 - d^2 = \frac{d^2}{2}$

Barrel width (diameter) of largest volume:

$d_{max} = R\sqrt{\frac{2}{3}}$

Barrel length of largest volume:

$l_{max} = 2\sqrt{R^2 - \frac{2R^2}{3}} = \frac{2R}{\sqrt{3}}$

# d - barrel width
# R - fixed rod length

R = 1

# barrel length function of d
length = lambda d:2*np.sqrt(R**2 - d**2)

# volume function of d
volume = lambda d:length(d)*(np.pi*d**2)/4

# barrel width (diameter) range with step = 0.01
d = np.arange(0,R,0.01)

# Barrel width (diameter) of largest volume:
d_max = R*np.sqrt(2/3)

# Barrel largest volume:
v_max = volume(d_max)

# Barrel length of largest volume:
l_max = length(d_max)

plt.figure(figsize = (10, 8))
plt.plot(d, volume(d), 'b', \
         label=f'barrel volume function of d')
plt.plot(d, length(d), 'r', \
         label=f'barrel length function of d')
plt.axvline(d_max, -10, 10, label=f'd = d_max = {d_max}')
plt.axhline(v_max, 0, 20, color = 'k', \
        label=f'v = v_max = {v_max}')
plt.plot([d_max],[l_max], 'ro', \
         label=f'd_max={d_max}, l_max={l_max}')
plt.plot([d_max],[v_max], 'bo', \
         label=f'd_max={d_max}, v_max={v_max}')
plt.xlabel('barrel width (d)')
plt.ylabel('Barrel Volume (v) / Length (l)')
plt.legend()
plt.title('Finding barrel dimentions of largest volume for fixed rod length (R = 1)')
plt.grid()
plt.show()

Barrel length as argument¶

Length range: L = 0 .. 2R

$d^2 = R^2 - \frac{l^2}{4}$

$d = \sqrt{R^2 - \frac{l^2}{4}}$

Volume as a function of barrel length:

$vol(l) = \frac{\pi d^2}{4}l = \frac{\pi R^2 l}{4} - \frac{\pi l^3}{16}$

The first derivative of vol(l):

$vol^{'}(l) = \frac{\pi R^2}{4} - \frac{3 \pi l^2}{16}$

Find barrel dimentions of largest volume ($vol^{'}(l) = 0$)

$\frac{\pi R^2}{4} = \frac{3 \pi l^2}{16}$

Barrel length of largest volume:

$l = \frac{2R}{\sqrt{3}}$

Barrel width (diameter) of largest volume:

$d = \sqrt{R^2 - \frac{R^2}{3}} = R\sqrt{\frac{2}{3}}$

Calibration Curve for barrels of optimal dimentions¶

Barrel volume as function of rod length

Barrel width (diameter) of largest volume:

$d_{max} = R\sqrt{\frac{2}{3}}$

Barrel length of largest volume:

$l_{max} = \frac{2R}{\sqrt{3}}$

Volume as a function of rod length:

$vol(d) = \frac{\pi d_{max}^2}{4}l_{max} = \frac{\pi R^3}{3\sqrt{3}}$
